# EOS M6 Mark II
This is current platform support for EOS M6 Mark II 1.1.1 ( **5.9.2** )

## Caveats
There are **TWO DIFFERENT VERSIONS** of 1.1.1 ROM: 5.9.2 and 5.9.3

Version 5.9.3 exists only pre-installed on cameras, there was no firmware update
file issued.

**You need to have a correct version installed for Magic Lantern to run.**
It is impossible to check internal version without running Basic scrips.
But there are no tricks needed to install older version even if camera has
5.9.3 installed from a factory.
